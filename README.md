# 数据可视化项目集 - 第五章

## 📊 项目简介

本项目基于第五章Jupyter Notebook的数据分析内容，使用现代Web技术构建了一个完整的交互式数据可视化平台。项目将传统的静态数据可视化转换为动态、交互式的Web应用，提供丰富的用户交互体验。

## 🎯 核心特性

### 🚀 交互式可视化
- **7种图表类型**：折线图、柱状图、饼图、环形图、水平条形图、混合图表
- **丰富交互**：缩放、平移、悬停查看详情、数据筛选
- **动画效果**：流畅的加载动画和交互过渡效果
- **响应式设计**：完美适配桌面、平板、手机等各种设备

### 📈 数据覆盖
- **产品销售数据**：产品A与产品B的12个月销售趋势对比
- **宠物统计数据**：12个国家养猫养狗人群比例分析
- **社交媒体分析**：抖音2017-2018用户增长和地区分布
- **气候数据**：温度、降水量、蒸发量的多维关系分析

### ⚙️ 技术特性
- **现代Web技术**：HTML5 + CSS3 + JavaScript (ES6+)
- **专业图表库**：Chart.js 4.x + Chartjs-plugin-zoom
- **无框架依赖**：轻量级实现，无需React/Vue等框架
- **模块化架构**：清晰的代码结构，易于维护和扩展

## 📊 数据详情

### 产品销售数据
- **时间范围**：12个月（1月-12月）
- **数据内容**：产品A和产品B的销售额（亿元）

- **地理范围**：12个国家
- **数据内容**：养猫和养狗人群比例（%）

### 抖音用户数据
- **时间对比**：2017年 vs 2018年
- **地区分类**：5个地区等级
- **分析维度**：增长倍数、地区分布变化

### 气候数据
- **时间范围**：12个月
- **数据内容**：温度（°C）、降水量（ml）、蒸发量（ml）
- **分析维度**：相关性分析、季节性变化

## 🛠️ 技术实现

### 核心技术栈
- **HTML5**：语义化标签、Canvas绘图
- **CSS3**：Flexbox/Grid布局、动画过渡、响应式设计
- **JavaScript**：ES6+语法、模块化编程、事件处理
- **Chart.js**：专业图表渲染、插件系统
- **Chartjs-plugin-zoom**：图表缩放平移功能

#### 1. 响应式布局

new Chart(ctx, {

## 📱 功能说明

### 主要功能
1. **图表交互**
   - 鼠标悬停显示详细数据
   - 滚轮缩放图表区域
   - 拖拽平移查看不同数据区间

2. **数据筛选**
   - 按图表类型筛选（折线图、柱状图、饼图等）
   - 按数据源筛选（产品、宠物、抖音、气候）
   - 实时筛选，无需刷新页面

3. **动画控制**
   - 可开关的加载动画
   - 图表更新过渡效果
   - 悬停动画反馈

4. **数据导出**
   - JSON格式导出所有原始数据
   - 保留完整的数据结构和元信息

### 快捷操作
- `Ctrl + R`：重置所有图表缩放
- `Ctrl + S`：导出数据文件
- `F12`：打开开发者工具查看调试信息

## 🔧 故障排除

### 常见问题

#### 图表不显示
1. **检查网络连接**：确保能访问CDN资源
2. **使用调试版本**：打开 `_调试版.html` 查看详细错误信息
3. **浏览器兼容性**：建议使用Chrome、Firefox、Safari、Edge等现代浏览器

#### 缩放功能异常
1. **启用缩放选项**：确保"缩放功能"复选框已勾选
2. **检查插件加载**：查看调试信息确认缩放插件加载状态
3. **刷新页面**：重新加载页面重置状态

#### 响应式问题
1. **清除浏览器缓存**：强制刷新页面（Ctrl+F5）
2. **检查视口设置**：确保meta viewport标签正确
3. **测试不同设备**：使用开发者工具的设备模拟

### 调试工具
项目提供了专门的调试版本，包含：
- 依赖库加载状态检查
- 详细错误信息显示
